# 📊 ConnectaTel - Análisis de Clientes

## Objetivo
Evaluar el comportamiento de los clientes de ConnectaTel, empresa de telecomunicaciones en LATAM, identificando patrones de consumo, comportamientos atípicos y segmentos de clientes.

## Datasets utilizados
- `plans.csv` — Información de los planes disponibles
- `users_latam.csv` — Información de los clientes
- `usage.csv` — Detalle del uso real de servicios

## Etapas del análisis
1. Carga y exploración de datos
2. Identificación de problemas de calidad de datos
3. Limpieza básica de datos
4. Summary statistics de uso por usuario
5. Visualización de distribuciones y outliers
6. Segmentación de clientes por uso y edad
7. Insight ejecutivo para stakeholders

## Cómo ejecutar
1. Abre el notebook en [Google Colab](https://colab.research.google.com/)
2. Carga los datasets en `/datasets/`
3. Ejecuta las celdas en orden de arriba a abajo

## Tecnologías
Python, Pandas, NumPy, Seaborn, Matplotlib
